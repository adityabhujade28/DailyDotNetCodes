# Daywise C# Projects

This repository contains a collection of small C# projects and daily coding tasks. Each folder represents a different project or application, created to practice and demonstrate various C# concepts, patterns, and features.


## Purpose
This repository serves as a personal archive of daily C# coding exercises, mini-projects, and learning experiments. It is intended for:
- Practicing C# programming skills
- Exploring .NET features and libraries
- Building small, functional applications
- Tracking progress and learning over time
